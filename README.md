# ColorPimp

Online color scheme generator for monotone and mixed colors.

## Getting Started

```bash
npm install
npm run serve
```

This compiles the SCSS and starts a development server at `http://localhost:9000`.

## Scripts

- `npm run serve` - Compile CSS and start dev server on port 9000
- `npm run build` - Build the project to the `dist/` directory
- `npm run build:css` - Compile SCSS to CSS only
- `npm run watch` - Watch SCSS files for changes and recompile
- `npm run clean` - Remove build artifacts
